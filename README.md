## Getting Started

Clone this repo and run 
```sh
yarn install
```
to install all the dependencies listed in `package.json`.

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.
